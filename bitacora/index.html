Mi Bitácora Semanal

Semana 01: Introducción al proyecto Innovador y a la Bitácora Digital

Se hizo una introducción al proyecto innovador. Se presentaron los alumnos y también dibujaron un vegetal para que sirva como un CV Visual. También se habló y se dio detalles para realizar nuestra Bitácora Digital. Utilizaremos el software de Visual Studio Code para la realización de la página web dónde utilizaremos código HTML, CSS y Javascript.

Semana 02: Repasando la sintaxis y estructura de una página web

Repasamos el tema de las páginas web y nos familiarizamos con las etiquetas HTML para poder diagramar y editar mejor la bitácora digital. Gestionamos mejor el contenido de página web para poder presentarlo próximamente y obtener feedbacks.

Semana 03: Accediendo a GitHub y desplegando la página web

Comenzamos con la introducción a Netlify. Luego se implementó el uso de GitHub , creacion de un repositorio y subida de archivos. Una vez realizado todo esto se puede desplegar el proyecto web para luego de esto poder ser visualizado en un hosting y dominio en la nube.

Semana 04: Profundizando las herramientas de OnShape

Se realizó un trabajo con mayor profundidad utilizando las herramientas de OnShape. Pudimos recrear una botella con líquido interior, así mismo se podían pintar sus caras, color de botella, color de tapa y otras cosas más.

Semana 05: Introducción a TinkerCAD

Nos iniciamos en el mundo de TinkerCAD que es una herramienta para diseñadores y creadores en donde podremos construir cosas como robots, casas en miniatura, etc. En nuestro caso estamos aprendiendo temas como circuitos, arduino, conexiones entre otras cosas.

Semana 06: TinkerCAD, motor en sentido horario y antihorario

En esta clase genial, nos zambullimos en el mundo de la programación de motores en Tinkercad! La idea es hacer que nuestro motor se comporte a nuestro antojo, ya sea girando felizmente en sentido horario o desafiando la gravedad en sentido antihorario.

Semana 07: TinkerCAD, sensores ultrasónicos

¡Vamos a sumergirnos en el fascinante mundo de Tinkercad y los sensores ultrasónicos en esta clase emocionante! Imagina tener la capacidad de medir distancias con precisión, ¡eso es exactamente lo que los sensores ultrasónicos nos permiten hacer!

Semana 08: TinkerCAD, prender y apagar LEDS con dos pulsadores

¡En nuestra aventura por Tinkercad, hoy exploramos el emocionante territorio de encender y apagar LEDs con dos pulsadores! Imagina tener el control total sobre la luz con solo presionar un botón. ¡Vamos a hacerlo realidad!

Semana 09: ¿Qué es un pseudocódigo?

¡En nuestra excursión por el vasto terreno de la programación, hoy nos sumergimos en el intrigante mundo del pseudocódigo! ¿Alguna vez te preguntaste cómo los programadores diseñan sus algoritmos antes de traducirlos a un lenguaje de programación específico? ¡Aquí es donde entra en escena el pseudocódigo!

Semana 10: Explicación para notas del promedio 03

Se nos explica cómo se divide la nota de promedio 03, notas de presentación, una tarjeta de realidad aumentada, pitch de presentación de producto, una bitácora personal y una página web de nuestro producto.

Semana 11: Introducción a ZAPWORKS

¡Hoy nos adentramos en el emocionante universo de ZAPWORKS! ¿Te imaginas poder crear experiencias de realidad aumentada sin ser un mago de la programación? ¡Pues bien, ZAPWORKS es tu varita mágica para hacerlo realidad!

Semana 12: Presentación de posibles proyectos

¡Hoy nos embarcamos en un viaje emocionante explorando las infinitas posibilidades de proyectos con Arduino! Imagina tener el poder de dar vida a tus ideas y crear dispositivos interactivos. ¡Con Arduino, el único límite es tu imaginación!
